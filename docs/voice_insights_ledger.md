# Voice Insights Ledger — 声の台帳

**Constitution** (VOICE_MISSION.md + listening_guide_ja.md §記録の規律): every entry is a real person's words, **verbatim**. A paraphrase presented as a quote is fabrication (OPS-062(D)/065 extended to humans — D4-severity). Design decisions cite entries by ID; a decision with no entry behind it is labeled 私たちの想定 (our assumption), honestly.

## Entry classes

- **LIVE-VOICE** — a driver speaking WITH us (the A6 conversation; a consented LINE voice memo; a partnered listening visit). Consent recorded twice: to record, and separately for quote-use. Machine transcripts are DRAFT until a human verbatim pass against the audio (dialect preserved as spoken; standard-ja gloss in brackets); only then VERBATIM-stamped with verifier + date. Deletion on request is absolute, including every design citation (the decision survives only relabeled 私たちの想定). **The Voice Task Force activates on LIVE-VOICE entry #1.**
- **PUBLIC-VOICE** — published words read where they already live (知恵袋, みんカラ, 市民の声, radio archives). URL + post date + retrieval date per entry; aggregate community framing; individuals only via their own published words. Informs the design vocabulary; does NOT activate the task force (listening is one-way — it cannot test OUR utterance).

## Entry format

```
### [LV|PV]-NNN · YYYY-MM-DD retrieved/heard
> 原文そのまま(ja)
— source: URL / conversation ref · post date · speaker label (consented name, or "知恵袋投稿者" class label)
context: one neutral line (no interpretation)
```
