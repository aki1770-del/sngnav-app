このアプリは、雪道の運転を支えるための情報アプリです。まだ**アルファ段階** — 育てている途中の道具です。完成品としてではなく、「一緒に確かめてくれる人」としてあなたに使っていただけることが、何よりの力になります。

## いちばん大事なこと（安全について）

**運転が最優先です。** アプリは、見ない・触らないでよいように、音と振動で知らせるつくりにしています。表示や動きに「おかしいな」と思うことがあれば、**無視してください**。運転の判断は、いつもあなた自身のものです。アプリは情報を出すだけで、運転の責任を代わりに持つことはできません。

## はじめる前のひと手間（音声の設定）

電波のない場所でも音声が出るように、端末に**日本語のオフライン音声データ**を入れておくことをおすすめします。

だいたいの手順（機種により名前が少し違います）:
「設定」→「テキスト読み上げ」（「ユーザー補助」や「一般管理」の中にあることが多いです）→ 音声データのインストール → 日本語

見つからなければ、そのまま教えてください。それ自体が大事な報告です。

## ドライブで試してみてほしいこと

無理のない範囲で、ふだんの運転のついでにで十分です。

1. **出発前にひと目** — 気温・湿度などの観測値（気象庁の値そのままです）と、警報・注意報、そして「路面凍結ウォッチ」の行を見てみてください。
2. **走行中の画面** — 現在地の点がちゃんとついてくるか、注意の表示が読みやすいか。画面は走行中に消灯しないようにしてあります。
3. **「路面凍結ウォッチ」「荒天ウォッチ」の行** — 表示の言葉がわかりやすいか、読み上げが聞き取れるか。データが足りないときは「判定不能」と正直に表示します — 「大丈夫」とは言いません。
4. **機内モードで地図** — 電波を切っても、秋田の地図が表示されるか試してみてください（地図は端末の中に入っています）。

## まだ確認できていないこと（正直に）

- **実機での音声の聞こえ方・振動の感じ方** — 仕組みの確認は済んでいますが、実際の端末で「運転中に聞こえるか・感じられるか」の確認はこれからです。あなたの報告が、まさにその確認になります。
- **凍結検知の本番確認** — このアプリの中心の約束（見えない路面凍結を知らせること）は、道路が実際に凍る**初雪のころ（2026年11月ごろ）**まで、本番の条件では確かめられません。それまでの間は、表示と使い勝手を一緒に整えていく期間です。
- **画面の一部はまだ英語です** — 日本語化を順に進めています。読めない・わかりにくい画面があれば教えてください。

## 「秋田のモック位置（開発用）」ボタンについて

現在地の共有ボタンの隣に「秋田のモック位置（開発用）」というボタンがあります。これは**開発者用**のボタンで、GPS の代わりに秋田駅の固定の位置を表示します。間違えて押しても大丈夫です — 開発用の位置は**琥珀色（オレンジ）の点**で表示され、本物の GPS（青い点）とは色も表記も区別されています。ふだんのテストでは押さなくてかまいません。

## 気づいたことの送り方

どんな小さな違和感でも価値があります。「音が鳴らなかった」「表示が変だった」「言葉がわかりにくい」「窓の外の天気と違う」— 全部、教えてください。

- **いちばん簡単な方法**: アプリの「**ログを共有**」ボタンを押す → 共有画面が開く → 写真を送るのと同じ要領で、いつもの連絡手段で送ってください。
- **もちろん、直接話してくださるだけでも十分です。** メモも形式もいりません。

送っていただくログは端末の中の不具合記録だけで、あなたが共有ボタンを押したときにだけ端末の外に出ます。自動で送られるものは何もありません。
